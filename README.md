# FlexiBudget App

## 📱 Overview
FlexiBudget is a prototype Android budgeting app built with Kotlin and XML layouts.  
It includes screens for:
- Login
- Home
- Expense
- Category
- Subscriptions
- Reports

## 🎨 Features
- Sky‑blue themed UI
- Navigation between prototype screens
- Logging for button clicks and activity lifecycle
